The position of links

Now we explore the effect the position of the link has on the probability of the user clicking it. There are many factors that can affect the probability of a player clicking a link. We use the actual shortest path to the link as an unbiased estimator of the probability of the player clicking the link. Therefore, if we compare the position of the link that the player actually clicks to the link that would get him closest to the target, we can see how the position of the link affects the probability of the player clicking it.

The shortest path links is uniformly distributed over the pages. However, users are biased towards links that appear higher on the page.
